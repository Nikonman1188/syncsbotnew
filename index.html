-say

Says what you say

Example: -say {message}

-ban

Bans a user

Example: -ban {@user} {reason}

-unban

Unbans a user

Example: -unban {user id}

-kick

Kicks a user

Example: -kick {@user} {reason}

-ping

Returns the bot ping

Example: -ping

-userinfo

Shows a user info

Example: -userinfo {@user}
